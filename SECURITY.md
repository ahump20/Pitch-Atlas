# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report vulnerabilities privately via GitHub Security Advisories for this repository.
Include:
- a clear description of the issue
- reproduction steps or proof of concept
- potential impact

We will triage reports as quickly as possible and coordinate disclosure and remediation.

## Scope Notes

Pitch Atlas v1 is a static, provenance-first visual proof. It does not include auth or runtime external API calls.
